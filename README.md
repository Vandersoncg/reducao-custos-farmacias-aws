# Redução de Custos em Farmácias com AWS

## Sobre o Projeto
Projeto prático desenvolvido na plataforma DIO com o objetivo de demonstrar o uso de serviços da Amazon Web Services (AWS) para redução de custos operacionais em uma empresa fictícia do setor farmacêutico.

## Serviços AWS Utilizados
- Amazon EC2 com Auto Scaling
- Amazon S3
- Amazon RDS

## Objetivo do Projeto
Elencar três serviços AWS capazes de promover redução de custos imediatos e melhoria da eficiência operacional.
